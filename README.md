# Cdower's Home Assistant Add-ons

A small collection of Home Assistant add-ons I maintain.

## Add this repository to Home Assistant

Click the button below, or copy the URL below into Settings → Add-ons → Add-on Store → ⋮ → Repositories.

[![Add to Home Assistant][add-repo-shield]][add-repo-url]

```text
https://github.com/Cdower/hassio-addons
```
